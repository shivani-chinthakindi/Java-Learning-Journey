# Java Basics

This folder contains programs covering the fundamental concepts of Java programming.

These concepts form the foundation for learning Data Structures, Object-Oriented Programming, and Software Development.

### Operators

- Implicit Type Casting
- Explicit Type Casting

### User Input and Output
- Scanner Class
- Reading Different Data Types

### Conditional Statements
- if
- if-else
- else-if ladder
- nested if

### Switch Statements
- switch-case
- break statement

### Loops
- for loop
- while loop
- do-while loop

### Methods
- Method Creation
- Parameters
- Return Types

### Method Overloading
- Multiple methods with same name
- Different parameter lists
