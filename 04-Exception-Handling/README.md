# Exception Handling in Java

This section contains programs demonstrating Exception Handling in Java.

Exception Handling helps create robust applications by handling runtime errors gracefully and preventing unexpected program termination.

## Folder Structure

- TryCatch
- FinallyBlock
- ThrowAndThrows
- CustomExceptions

## Concepts Covered

### Try-Catch
- Handling runtime exceptions
- Catching specific exceptions
- Multiple catch blocks

### Finally Block
- Resource cleanup
- Code execution regardless of exceptions

### Throw and Throws
- Manually throwing exceptions
- Declaring exceptions

### Custom Exceptions
- Creating user-defined exceptions
- Handling business logic errors
