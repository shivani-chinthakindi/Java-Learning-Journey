# Strings in Java

This folder contains programs and practice exercises related to Strings in Java.

## Topics Covered

- String Declaration
- String Input & Output
- String Methods
- String Comparison
- String Concatenation
- StringBuilder
- Reverse String
- Palindrome Check
- Counting Vowels
- Character Frequency

## Files

| File Name | Description |
|------------|-------------|
| StringDeclaration.java | Creating and initializing strings |
| StringMethods.java | Practicing commonly used string methods |
| StringComparison.java | Comparing strings using equals(), == and compareTo() |
| StringConcatenation.java | Combining multiple strings |
| StringBuilderDemo.java | Using StringBuilder for efficient string manipulation |
| ReverseString.java | Reversing a string |
| PalindromeCheck.java | Checking whether a string is a palindrome |
| CountVowels.java | Counting vowels in a string |
| CharacterFrequency.java | Finding frequency of characters in a string |
| StringPractice.java | Additional string practice problems |

## Learning Outcomes

After completing these programs, I will be able to:

- Create and manipulate strings
- Use built-in string methods effectively
- Compare strings correctly
- Perform common string operations
- Work with StringBuilder
- Solve beginner string-based problems
- Understand character frequency and pattern-based questions
