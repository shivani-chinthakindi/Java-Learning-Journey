# Arrays in Java

This folder contains programs and practice exercises related to Arrays in Java.

## Topics Covered

- Array Declaration
- Array Initialization
- User Input in Arrays
- Array Traversal
- Finding Maximum Element
- Finding Minimum Element
- Swapping Elements
- Reversing an Array
- Linear Search
- Binary Search
- Multidimensional Arrays

## Files

| File Name | Description |
|------------|-------------|
| ArrayDeclaration.java | Creating and initializing arrays |
| ArrayInputOutput.java | Taking input and displaying array elements |
| ArrayTraversal.java | Traversing arrays using loops |
| FindMaximumElement.java | Finding the largest element in an array |
| FindMinimumElement.java | Finding the smallest element in an array |
| ReverseArray.java | Reversing an array |
| SwapElements.java | Swapping elements in an array |
| LinearSearch.java | Searching an element using Linear Search |
| BinarySearch.java | Searching an element using Binary Search |
| MultiDimensionalArray.java | Working with 2D arrays |
| ArrayPractice.java | Additional practice problems |

## Learning Outcomes

After completing these programs, I will be able to:

- Create and initialize arrays
- Access and modify array elements
- Traverse arrays efficiently
- Implement searching techniques
- Solve basic array-based problems
- Work with multidimensional arrays
